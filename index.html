About this App

Portfolio ready cybersecurity demo

This is a static, browser based single page application designed for demos and portfolios. It intentionally uses simulated data so it is safe to publish publicly on GitHub Pages.

What it demonstrates

  • Dashboard KPIs and real time charts
  • Simulated vulnerability assessment workflow with findings and remediation tips
  • Password strength scoring with clear user feedback
  • Threat feed filtering, searching, and alert triage style UI
  • Streaming network visualization charts
  • Exportable report for screenshots and documentation

Safe use note

This app does not scan networks, ports, websites, or hosts. It does not collect data. It is a UI and analytics demo only.
